## Codes in responses

## Success

0 - 999

* 0 - success

## Invalid input codes

1000 - 1999

* 1000 - json not valid

* 1001 - email already exists
* 1002 - group name already exists


* 1501 - email does not exist

## Database errors

2000 - 2999

## Authentication errors

3000 - 3999

* 3001 - failed password hashing
* 3002 - failed password validation
* 3003 - not authorized to access group
